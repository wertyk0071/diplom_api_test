2ое задание

Из таблицы "Orders" выбрать колонку track (номер трекера) и вычислить статус по условиям:

Если finished = true, статус 2.

Если cancelled = true, статус -1.

Если inDelivery = true, статус 1.

Во всех остальных случаях статус 0.

При этом важен порядок проверки: если заказ одновременно и завершён, и отменён (что маловероятно, но всё же), то приоритет отдаётся finished (2), затем cancelled (-1), затем inDelivery (1). В жизни так: если заказ выполнен, он не может быть в доставке, поэтому проверяем сначала finished, потом cancelled, потом inDelivery.



 отмена заказа

    "track": 959324
}

SELECT
    track,
    CASE
        WHEN "finished" = true THEN 2
        WHEN "cancelled" = true THEN -1
        WHEN "inDelivery" = true THEN 1
        ELSE 0
    END AS status
FROM
    "Orders";
